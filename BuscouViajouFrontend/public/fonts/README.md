## Como o frontend referencia

As declarações `@font-face` ficam em `src/styles/tokens.css` e usam o caminho público `/fonts/gotham/<arquivo>.woff2` (servido a partir de `public/fonts/gotham/`). Todas as faces são declaradas sob a família `'Gotham'` com `format('woff2')` e `font-display: swap`. Exemplo do padrão usado:

```css
@font-face {
  font-family: 'Gotham';
  src: url('/fonts/gotham/gothambook-webfont.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
```

A família é então consumida pela variável `--bv-font-heading: 'Gotham', 'Helvetica Neue', Arial, sans-serif;` definida no mesmo `tokens.css`.

## Origem

Os arquivos `.woff2` deste diretório foram copiados do pacote `_context/gotham-fonts-main/web/` (vide `_context/gotham-fonts-main/web/stylesheet.css`, gerado por Font Squirrel em 12/01/2021). O `stylesheet.css` original desse pacote referencia também variantes `.woff` (não copiadas para `public/fonts/gotham/`) e usa caminhos relativos; no frontend, somente os `.woff2` são servidos, com caminhos absolutos `/fonts/gotham/...` declarados em `src/styles/tokens.css`.
